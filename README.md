Backend (Spring Boot API & Domain Driven Design): Implementation of a RESTful API based on a layered model (Controller, Service, Repository). The technical focus consisted of modeling a complex domain with JPA/Hibernate, managing states via Enums, and implementing ManyToMany associations with join tables and extra data. Persistence was optimized with custom exception handling for referential integrity and service-level calculated subtotal/total logic.

Tools used: Java 11+, Spring Boot, JPA/Hibernate, Maven, PostgreSQL, H2 Database, Postman, and Heroku.
